What it does

Everything your server needs

🪙

Economy

Coins, Gems, and Tokens. Work, gamble, rob, and trade your way up the leaderboard.

🎮

Games

Trivia, Hangman, Wordle, Blackjack, Tic-Tac-Toe, Connect Four, and more — all earning real rewards.

🎣

Fishing

18 fish types across 6 rarity tiers. Upgrade your rod and compete on the fishing leaderboard.

⌨️

Fun Games

Typerace, riddles, Would You Rather, Emoji Decode, Hot Potato, Word Chain — new ways to earn every day.

🛡️

Moderation

Full mod suite: warn, kick, ban, timeout, purge, slowmode, lock, nick, and a full case log.

🏆

Levels & Roles

XP per message, milestone roles at levels 5/10/20/30/50, and multipliers per subscription tier.

🎁

Giveaways & Polls

Create timed giveaways with multiple winners, community polls, and scheduled reminders.

🛒

Premium Shop

24+ items across three currency shops. XP boosts, fishing rods, streak shields, lucky gambles, and more.

Command reference

All commands, prefix t!

t!balance [@user]

View coins, gems, and tokens

t!daily

Claim daily coins + streak bonus

t!work

Earn coins (1h cooldown)

t!gamble <amount>

Gamble coins (45% win chance)

t!slots <bet>

Slot machine — jackpot pays gems

t!rob @user

Attempt to rob another user

t!transfer @user <amount>

Send currency to someone

t!shop [coins|gems|tokens]

Browse the premium shop

t!buy <item_id>

Purchase a shop item

t!inventory

View your active items

t!richest

Server wealth leaderboard

t!joke [category]

Get a joke — dad / puns / tech / animal / food

t!story [genre]

Get a story — adventure / mystery / fantasy

t!trivia [diff]

Live trivia — easy / medium / hard (Vibe+)

t!hangman [diff]

Hangman word game (Premium+)

t!wordle

Wordle-style word game (Premium+)

t!blackjack <bet>

Blackjack vs dealer (Pro only)

t!rps

Rock Paper Scissors with buttons

t!guess

Number guessing game

t!ttt

Tic-Tac-Toe vs bot

t!c4

Connect Four vs bot

t!scramble

Unscramble a word — channel race

t!mathquiz [diff]

Rapid-fire math quiz

t!highlow [bet]

Higher or Lower card game

t!duel @user <bet>

Coin flip duel vs another user

t!snap

Reaction speed game

t!typerace

Type a sentence fast — coins based on speed

t!riddle

First correct answer wins 80 coins + 1 gem

t!wouldyourather

Vote with buttons — all voters earn 1 token

t!emojidecode

Decode an emoji phrase for coins and tokens

t!fastmath

Channel race math question — speed bonus coins

t!hotpotato

Pass the button — holder when timer explodes loses coins

t!wordchain

Chain words — 20 coins per valid word, gems for top scorer

t!fish

Cast your line (15s base cooldown)

t!fishbag

View your current fish inventory

t!sell

Sell all fish for coins

t!fishstats

View your fishing statistics

t!fishtop

Top fishers leaderboard

t!buy rod_silver

Silver Rod — +30% uncommon / +50% rare

t!buy rod_gold

Gold Rod — +60% uncommon / +120% rare

t!buy rod_diamond

Diamond Rod — +200% legendary + shorter cooldown

t!warn @user <reason>

Issue a warning (logged)

t!warnings @user

View a user's warning history

t!kick @user [reason]

Kick a member

t!ban @user [reason]

Ban a member

t!unban <user_id>

Unban a user by ID

t!timeout @user <duration>

Mute a user temporarily

t!purge <amount>

Bulk delete messages

t!slowmode <seconds>

Set channel slowmode

t!lock / t!unlock

Lock or unlock a channel

t!modlog

View recent moderation cases

t!setmodlog #channel

Set the mod log channel

t!nick @user <name>

Change a member's nickname

t!level [@user]

Check XP and current level

t!rank

Server XP leaderboard

t!profile [@user]

Full profile card with tier badge

t!leaderboard

Auto-updating top richest, XP, and fishers

t!help [category]

Full command list with dropdown

t!ping

Check bot latency

t!stats

Bot statistics

t!daily

Claim daily coins

t!subscribe

View subscription tiers and pricing

t!upgrade <tier>

Upgrade your subscription

t!remind <time> <msg>

Set a personal reminder

t!poll <question> | <opt1> | ...

Create a community poll

t!gcreate

Create a giveaway

t!serverinfo

Server statistics
